
Send String is a simple, no-friction tool that lets you scan barcodes with your phone and instantly send the decoded text to your computer browser.

No cables. No apps. No typing long codes manually.

Just open the page, scan the QR code to pair your phone, and start scanning — the results appear in real time on your desktop.

Perfect for warehouses, labs, workshops, or any workflow where barcode scanners are limited but smartphones are everywhere.

Key features:
Instant pairing via QR code
Real-time barcode transfer to browser
No installation required (works in browser)
Lightweight and open-source
Designed for speed and simplicity
Stop typing. Start scanning.
